# mhrise-monster-info
A mod for Monster Hunter Rise that shows information about the nearest large monster

## Features
* Designed to work without reframework-d2d
* Show HP
  * Blue if catchable
* Show ailments
* Configurations:
  * Position
  * If enabled
  * If enabled online
  * What ailment should be shown
  * Show ailment if > 0

## Requirements
* Reframework
